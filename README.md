# NGIT Frontend Week 2 - Digital Professional Bio

## Project Description
This is my Week 2 frontend assignment for the NGIT training program.
It is a single-page digital professional bio built with pure HTML5,
showcasing my background, skills, education, and contact information.

## Tags I Used

<header>For the page header section
<main> For the main content area
<footer>For the footer section
<section> For grouping related content
<h1> to <h2> For headings
<p> For paragraphs
<em> For italic text on goals
<ul> and <li> For unordered skills list
<ol>and <li> For ordered education/experience list
<img> For profile photo with alt attribute
<a> For GitHub link and email (mailto) link
<strong> For bold text on key skills

## Tags I Found Most Useful
I found <section>, <header> <main>, and <footer> most useful
because they make the page structure clear and readable for both
browsers and humans — which is what semantic HTML is all about.

## What I Learned
I learned how to use semantic HTML tags to properly structure
a webpage, how to add images with alt attributes, how to create
mailto links, and how to open links in a new tab using target="_blank and I learnt how HTML tags work together to build a webpage
and how block elements like div and p take up a full line,
while inline elements like a and span stay on the same line.
